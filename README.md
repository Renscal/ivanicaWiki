# Ivanica

Ivanica is a server-side Fabric mod for Minecraft 1.21.11 that adds survival/PvP mechanics: downed states, handcuffs, chains, a sleeping-body system for logged-out players, territory claiming, and a primed-TNT throw. No client mod required.

## Mechanics

| Mechanic | Description |
|---|---|
| [Down Mode](mechanics/down-mode.md) | Players are downed instead of killed — crawl, bleed out, or get revived |
| [Handcuffs](mechanics/handcuffs.md) | Cuff a downed player; restricts them, locks their inventory, and lets you muzzle them |
| [Chains](mechanics/chains.md) | Tether a cuffed player to yourself with an iron chain and drag them around |
| [Sleeping](mechanics/sleeping.md) | Logged-out players leave a sleeping body behind that can be killed |
| [Nations](mechanics/nations.md) | Form nations, claim territory, and protect it from outsiders |
| [TNT](mechanics/tnt.md) | Right-click with TNT to throw a primed explosive instead of placing a block |
